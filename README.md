# 📧 Mailvora – AI Email Agent System

Mailvora is an intelligent email processing agent that reads, analyzes, and classifies emails automatically using a modular backend system.

---

## 🚀 Project Overview

Managing emails manually is time-consuming. Mailvora introduces an AI-based agent that can:

- Read incoming emails  
- Understand their content  
- Classify them into meaningful categories  
- Serve as a base for future automation like smart replies  

---

## 🧠 Features

- 📥 Email loading from JSON dataset  
- 🤖 Email classification using agent logic  
- 🧩 Modular backend architecture  
- ⚡ Simple and efficient processing
